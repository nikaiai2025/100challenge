## 1. 追加AAキャラクター

### 1-1. やきう民 / やきうのお兄ちゃん 彡(ﾟ)(ﾟ)

- **発祥**: なんでも実況J板（なんJ）。2009年の野球ch → なんJ大規模移住の際に誕生
- **経緯**: 野球chからなんJへ移住してきたユーザーを「やきう民」と呼ぶ。もともとなんJに住んでいたユーザーは「原住民」（`(´・ω・`）`）と呼ばれ区別された。移住の最中、原住民の断末魔に初登場し、その後板の主流キャラとして定着
- **外見**: 黄色い肌、飛び出した目、尖った口。「彡」は髪の毛を表現
- **バリエーション**: 通常 `彡(ﾟ)(ﾟ)` / 喜び `彡(^)(^)` / 悲しみ `彡(;)(;)` / 無表情 `彡(-)(-)` 等
- **文化的展開**: 「彡(ﾟ)(ﾟ)で学ぶスレ」シリーズで教育系コンテンツとしても展開。やる夫スレの後継的な「なんJ民AAで学ぶ」ジャンルを形成
- **海外での受容**: 4chanに投稿された際、シンプソンズのキャラクターに似ていることから「Gumbly」「Graggle Simpson」の名で広まった

```
　 　 彡(ﾟ)(ﾟ)
```

### 1-2. ドクオ ('A`)

- **発祥**: 2002年2月頃、2ちゃんねる独身男性板（毒男板）
- **名前の由来**: 「独身男性」→「独男」→「毒男」→「ドクオ」
- **性格**: 常に無気力かつ消極的。独り身の人生を憂い、隣人のリア充生活を壁越しに聞いては虚無感に苛まれる。自殺を考えるが実行できない
- **口癖**: `('A`)ｳﾂﾀﾞｼﾉｳ`（鬱だ死のう）
- **文化的展開**: ブーン系小説でやる夫・内藤ホライゾンと並ぶ主要キャラとして活躍。ニュー速VIP板やゲーム系板でも広く使用された

```
('A`)ｳﾂﾀﾞｼﾉｳ

### 1-4. さいたま ( ﾟ∀ﾟ)

- **発祥**: 2001年～2002年頃。自民党のシンボルマーク「太陽を仰ぐ二人の子供」のAAをパロディ化した「自民党モナー」から派生
- **由来の諸説**: (1) 2001年5月のさいたま市誕生に影響を受けた説、(2) 2002年頃の最萌トーナメントで優勝キャラが意味もなく「さいたま!」と叫んでいたものが定着した説
- **特徴**: 3体のキャラクターが「さいたま！さいたま！」と意味もなく叫び続ける。中央の太陽にあたる丸い顔がFlash動画で最も多く使用された
- **Flash動画**: 音楽に乗って「さいたまさいたま」と言い続ける動画が大流行し、AA → Flash動画 の典型的な展開をたどった
- **派生キャラ**: 「みえ」「いばらき」「やまなし」「しずおか」「かごしま」等の都道府県バリエーションが多数存在

```

### 1-5. ヒッキー (-_-)

- **発祥**: 1999年、あやしいわーるど。モナーより古く、AAキャラクターの元祖の一つとされる
- **本名**: 小森マサオ
- **モデル**: 引きこもり。宇多田ヒカル（愛称ヒッキー）と同年齢のひきこもり青年という設定
- **外見**: 顔文字 `(-_-)` に下半身を付けたデザイン。部屋の隅で膝を抱えてうずくまった姿が基本
- **性格**: 万事悲観的。口癖は半角カナで「ｳﾂﾀﾀﾞｼﾉｳ」。よく首吊り自殺を図るが成功しない
- **ペット**: ちびギコのひきこもりバージョン「グレちゃん」
- **家族設定**: 同じ姿の遺影は父親（同じくひきこもりで他界）。母親も1年前に死亡
- **代表作**: 2000年暮に発表されたストーリーAA「ヒッキーのクリスマス」で一躍脚光を浴びた
- **ひきこもり板との関係**: 2ちゃんねるひきこもり板の名無し名が `(-_-)さん` に設定されている
- **現状**: NEETという概念に取って代わられ、ネット上であまり用いられなくなった

## 2. 偉人AA文化

### 概要

### 特徴

- 架空の人物のため、生没年が異常に短命だったり異常に長寿だったりする
- 嘘の解説文が付けられることも多い
- 2020年時点でほぼ新作は出ていない

## 3. 2ch全AA人気トーナメント（2002年）

### 概要

- **開催時期**: 2002年秋（予選は9月1日開始、約3ヶ月間）
- **趣旨**: 2ちゃんねるの全AAキャラクターの中から最も人気のあるキャラを投票で決定するイベント
- **背景**: 同年春～夏に開催された「第1回2ch全板人気トーナメント」の熱気を受けて企画された

### 結果

- **優勝**: ギコ猫 --- 四大AA（モナー・モララー・しぃ・ギコ猫）の中で、他の3体を直接対決ですべて撃破して頂点に立った
- **準優勝**: ニダー --- 東亜系キャラながら「リアルの韓国は嫌いだけどAAのニダーは好き」という票を集め善戦

### 全板人気トーナメントとの関係

2ちゃんねる全板人気トーナメントは板単位の人気投票で、第1回（2002年）から第3回まで開催された大規模横断イベント。

| 回 | 開催期間 | 優勝板 | 準優勝板 |
|----|----------|--------|----------|
| 第1回 | 2002年4月～7月 | Download板 | モ娘(狼)板 |
| 第2回 | - | ニュー速VIP板 | 自作PC板 |
| 第3回 | - | ネトゲ実況板 | モ娘(狼)板 |

トーナメント期間中は各板が支援AAを大量に制作し、AA文化の活性化に大きく貢献した。

## 6. AA改変文化（コラAA）

### 概要

既存のAAを改変・合成して新たな作品を生み出す文化。2ちゃんねるAA文化の根幹を成す創作手法であり、キャラクターの共有財産性を前提としている。

### 主な改変パターン

| パターン | 説明 | 例 |
|----------|------|----|
| 表情差分 | 同一キャラの表情を変える | ショボーン → シャキーン |
| キャラ合成 | 複数キャラの要素を組み合わせ | やる夫の顔 + 別キャラの体 |
| パーツ除去 | 既存AAからパーツを取り除く | やる夫 → できる夫（鼻を除去） |
| 時事改変 | 時事ネタに合わせてセリフや状況を改変 | 選挙・地震・スポーツ等 |
| 板固有改変 | 各専門板の文脈に合わせた改変 | ν速→ニュッ、嫌儲→けんもう君 |
| 巨大化 | 元のAAを拡大・精緻化 | 巨大モナー、巨大ギコ等 |

### AA改変の3000コマ超え

AA長編板では、コマ漫画のように複数の連続投稿でストーリーを構成する文化が発展。一部の作品は3000コマを超える超長編に達した。

### 改変文化が生んだ問題

- タカラ「ギコ猫」商標騒動、avex「のまネコ」騒動は、いずれも「改変元の共有財産性」が争点となった
- AAは誰か特定の個人が著作権を主張できない共有創作物であるという認識が、騒動を通じて社会的に定着

## 7. 時事ネタAA

2ちゃんねるでは重大な出来事が起きるたびに、専用のAAが即座に制作・改変された。

### 定番の時事ネタAA

| カテゴリ | 代表的なAA・使用場面 |
|----------|----------------------|
| **地震** | 揺れるモナー、「地震キタ━━━(ﾟ∀ﾟ)━━━!!」、倒壊する建物AA |
| **選挙** | 投票を呼びかけるAA、当選・落選のリアクションAA |
| **祭り** | 「祭りだ♪突撃だ」AA、夏祭り・神輿AA |
| **スポーツ** | サッカー・野球の試合実況AA、勝利・敗北のリアクション |
| **年末年始** | 年越しカウントダウンAA、正月AA |

### 実況文化との関係

実況板では試合やイベントの展開に合わせてリアルタイムでAAが投稿される。特に「キタ━━━(ﾟ∀ﾟ)━━━!!」は時事ネタAAの最も汎用的な形式であり、あらゆる速報に対して使用された。

- **概要**: さいとう・たかを原作の漫画『ゴルゴ13』の主人公デューク東郷をAA化したもの
- **用途**: 2ちゃんねるでは「狙撃」「暗殺」「プロフェッショナル」を表現する定番AA。スレッドの終了宣言（スナイプ）や、特定のレス番号を狙い撃つ際にも使用された

## 10. AA黄金時代の年表（詳細版）

| 年 | 出来事 |
|----|--------|
| 1999年 | 顔文字板設立（10月25日）。ヒッキー誕生（あやしいわーるど発） |
| 2000年 | モナー板設立（6月8日）。四大AA（モナー・モララー・ギコ猫・しぃ）の基本形が確立 |
| 2001年 | ぃょぅ誕生（5月5日）。ニダー誕生（3月15日）。シナー誕生（3月29日）。さいたま市誕生に合わせさいたまAA登場 |
| 2002年 | AA長編板設立（2月13日）。タカラ「ギコ猫」商標騒動（6月）。全AA人気トーナメント開催（秋、ギコ猫優勝）。AAサロン板設立（12月22日） |
| 2003年 | 『2ちゃんねる AA大辞典』刊行（3月）。でっていう誕生（10月）。AA長編板の黄金期 |
| 2004年 | ニュー速VIP板設立。内藤ホライゾン（ブーン）誕生。「電車男」ブームでAAが一般に認知 |
| 2005年 | avex「のまネコ」騒動。ちんぽっぽ誕生（8月29日）。ニダー板設立 |
| 2006年 | 2ch全AAイラスト化計画開始 |
| 2007年 | やる夫スレ文化の爆発的発展。嫌儲板新設（12月） |
| 2008年 | やる夫派生キャラ（やらない夫・できる夫・ドクオ等）の確立 |
| 2009年 | やる夫AA録公開（3月17日）。野球ch→なんJ大規模移住、やきう民誕生 |
| 2010年 | AA長編板サーバー故障で全スレ消失（9月9日）。やる夫AA録更新停止 |
| 2012年 | ニュー速→嫌儲大量移住。ニュッからけんもう君への世代交代 |
| 2016年 | 蝸牛くも『ゴブリンスレイヤー』刊行（やる夫スレ→商業ラノベの成功例） |
| 2019年 | KADOKAWA「スレ発ラノベ」4作品刊行 |
| 2020年代 | YouTubeショート・TikTok経由で小学生がAAキャラを認知する新現象 |
