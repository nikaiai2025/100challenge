# 2ちゃんねる/5ちゃんねる AA（アスキーアート）文化 追加調査レポート

## 1. 追加AAキャラクター

### 1-1. ショボーン (´・ω・`)

- **発祥**: 1999年頃からインターネット上で使用されていたが、2001年頃から2ちゃんねるで多用され広く知られるようになった
- **特徴**: がっかりした気分や気落ちした残念な気分を表現する顔文字・AA。「インターネット上で一番多く使われている顔文字」との噂があるほど汎用性が高い
- **用途**: 文末に付けて落胆を表現（例：「今日彼女に振られたよ(´・ω・`)」）
- **備考**: 顔文字としてもAAとしても使用される珍しいキャラクター。AAサロン板ではほぼ毎日新作が作られている

```
　　　　 (´・ω・`)
　　　　／　　　　＼
　　 ⊂　 )　　ノ＼つ
　　　　 (_⌒ヽ
　　　　　 ヽ ヘ ）
　　　　　　ﾉﾉ `Ｊ
```

#### 対義キャラ: シャキーン (`・ω・´)

- **特徴**: ショボーンの逆で、やる気に満ちた状態・気合が入った状態を表現
- **顔文字**: (`・ω・´)ｼｬｷｰﾝ
- **用途**: 決意表明や気持ちの切り替え時に使用

### 1-2. ニダー <丶`∀´>

- **誕生日**: 2001年3月15日

- **人気**: 2002年の2ちゃんねる全AA人気トーナメントで準優勝。「リアルの韓国は嫌いだけどAAのニダーは好き」という層も多く「東亜のアイドル」的存在

## 2. 定番顔文字・表現系AA

### 2-1. キタ━━━━(ﾟ∀ﾟ)━━━━!!

- **元ネタ**: 俳優・織田裕二が90年代後半のサンテFXのCMで目薬の爽快さをアピールして「キター！」と絶叫したこと
- **AA化**: 確認できる最古のスレはラウンジ板のもので、ラウンジ板の住人が現在の形を作ったと考えられている
- **普及のきっかけ**: ドラマ・映画にもなった「電車男」で多用されたことが決定的
- **用途**: 何かが登場・到来した際の驚きや期待を表現
- **派生**: ━の数を増減させたバリエーションが無数に存在

```
ｷﾀ━━━━━━(ﾟ∀ﾟ)━━━━━━ !!!!!
```

### 2-2. m9(^Д^) プギャー

- **前身**: 古くは「プゲラ（pgr）」「プゲラッチョ」「プゲラウヒョー」が使用されていた
- **成立**: 2004年のニュース速報VIP板設立あたりに現在の形に
- **構造**: 「m」は小指・薬指・中指、「9」は人差し指を表し、人を指差して嘲笑している様子
- **用途**: 相手を煽ったり嘲笑する場面で使用。「ざまぁ」と同義

```
m9(^Д^)ﾌﾟｷﾞｬｰ
```

## 3. AA長編板と名作AA作品

### 3-1. AA長編板の歴史

- **設立**: 2002年2月13日
- **目的**: AAで作るストーリー・物語の専門板
- **ルール**: 作品投稿中に割り込んで書き込みをすることは厳禁

#### スレッド形式

| 形式 | 説明 |
|------|------|
| ワンマンスレ | 単一作者による作品投稿（主流） |
| リレースレ | 複数作者が協力してストーリーを展開 |

#### 代表的な作品・スレッド

- **「世界制服を企むモララー」**: AA長編板の老舗リレースレ。複数の作者が参加する典型的なリレースレッド
- **「山奥のしぃ先生」**（2002～2003年、完結）: 通称「閣下」と呼ばれるAA職人による名作
- **「しぃのアトリエ」**（2005年～）: 同じく閣下による長期連載作品
- **「でぃちゃん死出の旅」**: 虐待されたしぃの派生キャラ「でぃ（D）」が東にある楽園を目指す物語

#### 衰退

- **2010年9月9日**: サーバーハードディスク故障により全スレッドが消失。スレッド数は100程度に激減
- **衰退要因**: pixiv、ニコニコ動画など、より簡単に創作物を発表できるプラットフォームの台頭

### 3-2. 虐殺系AAの歴史

2ちゃんねるAA文化のダークサイドとして、虐殺系AAという特異なジャンルが存在した。

#### 黎明期（2000-2001年）

- 初期の3大被虐AA: カメモナー、ビーグル、ぃょぅ
- 2001年10月: 「ひたすら虐殺・虐待」スレが立ち、しぃとちびギコが虐殺対象に

#### 虐殺ブーム期（2002年2月～2003年9月）

- 2002年2月末: 「しぃ虐待・虐殺スレッド」が立ち、月々約150作品が投稿される大量生産時代に
- モナー板は「2ちゃんねるで一番近づきたくない板」と言われるほどの状態に
- 2002年5月: 議論の末、したらば掲示板に旧アブ板が設立され、虐殺派の職人がモナー板から移動

#### しぃの変容

- 初期: 無口で感情表現が薄いキャラクター
- 虐殺ブーム期: 「泣き叫ぶ」「ダッコをねだる」など感情豊かなキャラクターへ進化
- **「アフォしぃ」**: 他者への攻撃性・自己中心的な性格を持つ「偽物のしぃ」として区別されるようになった

## 4. AA制作技術とツール

### 4-1. AAエディタの発展

#### Ascii Art Editor

- (´д`)Edit登場以前の定番ソフト
- 基本的なAA作成機能を備えていた

#### (´д`)Edit

- **開発言語**: Delphi 6
- **特徴**: Ascii Art Editorの上位互換。タブ機能、テンプレート機能、トレース機能を強化
- **地位**: AA職人・やる夫スレ作者から最も支持されたエディタ
- **派生ソフト**:
  - **(´д`)EditLite**: 機能を削った初心者向け・軽量版
  - **(´д`)Auto**: オートトレーサー（画像からAAを自動生成）
- **拡張**: 多数のスクリプトやツールが開発されている
- **現状**: 公開元ページは2017年3月1日からサーバー障害でアクセス不可だが、避難所で入手可能

#### その他のツール

- **シンプルAAエディタ on the WEB**: Webブラウザ上で動作するAAエディタ

### 4-2. フォントとズレ問題

2ちゃんねるのAAは **MS Pゴシック 16ピクセル（12ポイント）、行間2ドット** を標準として作られている。これはInternet Explorer 5.0での表示を基準としたものである。

#### ズレが起きる原因

- AAはプロポーショナルフォント「MS Pゴシック 12pt」を前提に作成されている
- UNIX、Mac、スマートフォンの固定幅フォントで見るとずれる
- Windows以外のOSにはMS Pゴシックが搭載されていない

#### 対策技術

| 対策 | 説明 |
|------|------|
| **モナーフォント** | MS Pゴシックの文字幅に合わせて作られたオープンソースフォント。Windows以外でもAAをズレなく表示可能 |
| **CSS設定** | `font-family: 'ＭＳ Ｐゴシック','MS PGothic',sans-serif;`、`white-space: pre;`、`line-height: 1.1;` の組み合わせ |
| **Textarフォント** | モナーフォントの後継的な位置づけのWebフォント |

### 4-3. MLTファイルとAA管理

- **MLTファイル**: (´д`)Edit用のAA管理ファイル形式。複数のAAをカテゴリ別に整理・共有するためのフォーマット
- **やる夫AA録**: 2009年3月17日公開のMLTファイル共有サイト。2010年4月26日にファイルサイズの肥大化と管理負荷により更新停止
- **やる夫スレ用MLT収集所**: やる夫AA録の後継。複数の管理者により2010年以降運営
- **AAMZ Viewer**: 2015年8月1日公開。MLT収集所のまとめzipをWeb上で閲覧できるビューアー

## 6. 商標問題の詳細

### 6-1. タカラ「ギコ猫」商標騒動（2002年）

| 日付 | 出来事 |
|------|--------|
| 2002年3月12日 | タカラ（現タカラトミー）がギコ猫のAAを対象に商標登録出願（商願2002-19166） |
| 2002年6月2日 | ニュース速報板に「【商標】『ギコ猫』はタカラの猫？【申請中】」スレッドが立ち、ユーザーが認知 |
| 2002年6月3日 | ひろゆき（西村博之）がタカラへ抗議文を送付。タカラが商標登録出願を取り下げ |

- **結末**: タカラは「ひろゆき氏と2ちゃんねるユーザーには真摯に対応し、お詫びしたい」と表明。発覚から解決まで約1日の電撃的終結
- **副産物**: この事件をきっかけに陽気で能天気なキャラ「タカラギコ」が誕生
- **意義**: ネット上の共有文化財産を企業が商標登録することの是非が初めて大きく問われた事件

### 6-2. avex「のまネコ」騒動（2005年）

#### 前史

- **2004年10月**: ルーマニアのO-Zoneの楽曲「恋のマイアヒ」の空耳Flash「マイヤヒー」が大人気に。Flash作者のわた氏がモナーとモララーが踊る動画を作成

#### 騒動の経緯

| 日付 | 出来事 |
|------|--------|
| 2005年9月1日 | avex子会社が「のまネコ」という商用キャラを作成し、グッズ展開開始 |
| 2005年9月22日 | 有限会社ZENが「のまネコ」の文字商標・図形商標を出願（avexと関連） |
| 2005年9月30日 | avexが商標登録を中止すると発表 |
| 2005年10月12日 | avexがキャラクター使用料を放棄すると表明、事態収束へ |

- **批判の焦点**: 「誰の物でもないネット上の既存キャラクターを改変して利益を得ようとしている」
- **タカラとの対比**: タカラは発覚翌日に謝罪・取り下げしたのに対し、avexは詫びるフリをしつつ商品化を進めたため批判が激化
- **犯罪予告事件**: 騒動に関連して複数の殺害予告が書き込まれ、逮捕者が出た

### 6-3. 商標問題の教訓と現在

- これらの事件は、ネット上の共有創作物（パブリックドメイン的な文化財）に対する権利問題の先駆的事例となった
- 2022年の「ゆっくり茶番劇」商標登録騒動にも同様の構図が見られ、20年以上同じ問題が繰り返されている

## 7. 2ch以外でのAA文化

### 7-1. ニコニコ動画との関係

- 初期のニコニコ動画文化はVIPPERなど2chユーザーが形成
- AAキャラクターを使った動画が多数投稿された
- 「ゆっくりしていってね!!!」（東方Projectキャラの饅頭AA）はニコニコ動画で爆発的に普及
- AA長編板の衰退要因の一つがニコニコ動画など動画プラットフォームの台頭

### 7-2. Twitter・SNSへの影響

- なんでも実況J板発の「ンゴ」「すこ」等のスラングがTwitter・TikTokを通じて若者文化に浸透
- 顔文字（ショボーン等）はLINEスタンプ化されるなど商品展開も
- AA自体のSNS投稿は、プロポーショナルフォント前提のため表示崩れが起きやすく衰退傾向

### 7-3. 現在のAA文化

- **YouTubeショート**: 2ch(5ch)まとめサイトの内容がYouTubeショートとして大量にアップロードされ、AAが動画内で使用されている
- **小学生への浸透**: 現在の小学生がYouTubeやTikTok経由でAAキャラクターを「知ってる有名キャラ」として認識している
- **AA職人の減少**: PCユーザーの減少、動画サイトの隆盛により、2000年代後半から職人数は継続的に減少
- **技術的障壁**: スマートフォンではMS Pゴシックが使えずAAが崩れるため、閲覧・制作ともにハードルが上がっている
