## AA文化の歴史と概要

アスキーアート（ASCII Art、略称AA）とは、文字や記号を組み合わせて絵を描く表現技法である。日本のインターネット文化においては、2ちゃんねる（現5ちゃんねる）を中心に独自のAA文化が発展した。

### 歴史的経緯

- **1999年10月25日**: 2ちゃんねるに「顔文字板」が新設。AAを扱う最初の専門板
- **2000年6月8日**: 「モナー板」（正式名称「モナー大好き@2ch掲示板」）がmentaiサーバーに新設。AAキャラクターの短編作品を扱う板として誕生
- **2002年2月13日**: 「AA長編板」が新設。長編AA作品の専門板
- **2002年頃**: 全盛期には2500ほどの書き込みがモナー板に集まったが、その後衰退傾向に

### AA文化の特徴

- 日本語環境が2バイト文字（全角文字）を扱えるため、半角のみの欧米AAより表現力が格段に高い
- 「AA職人」と呼ばれる職人的な制作者が数多く存在し、精緻な作品を生み出した
- キャラクターのAA化、Flash動画への展開、スレッド文化との融合など多方面に発展
- 「四大AA」としてモナー・モララー・ギコ猫・しぃが特に有名
- 英語圏では「Shift_JIS art」として認識され、MS PGothic前提の字幅で崩れずに見える点が日本AA文化の大きな特徴として紹介されている

## AA制作技術

### MS Pゴシック前提の文字幅設計

2ちゃんねる系掲示板のAAは、以下の表示環境を前提に設計されている。

- **書体**: MS Pゴシック（プロポーショナルフォント）
- **文字サイズ**: 16px（12pt）
- **行間**: 2px

MS Pゴシックはプロポーショナルフォントであるため、文字ごとに幅が異なる。AA職人はこの文字ごとのピクセル幅を把握した上で、上下の行で線が繋がるようにフォントを配置する。

### 文字幅の計算

AA職人「いのっく」の証言によれば、フォントの横幅は半角3dot〜全角16dotの範囲にあり、これらを計算して組み合わせることで立体感や曲線を表現する。1ドット単位の調整が求められる精密な作業である。

| 分類 | 文字幅 | 用途 |
|------|--------|------|
| 全角文字（漢字・かな） | 約16dot | 面の塗り、密度の高い部分 |
| 半角文字（英数・半角カナ） | 約8dot（全角の半分） | 細い線、微調整 |
| 半角スペース | 約4dot | 微細な位置調整 |
| 全角スペース | 約8dot | 大きな空白 |

### 制作プロセス

1. **目から描き始める**: 顔AAでは目が最重要パーツ。瞼の輪郭、瞳のハイライト、虹彩を意識して文字を配置する
2. **曲線の表現**: 右下がりの斜線（「\」方向）は文字の選択肢が多いが、右上がりの斜線（「/」方向）は適切な文字が少なく、初心者の最初の壁となる
3. **省略と補完**: 細部は「省ける部分はとにかく省く」。人間の視覚が自然に補完する性質を利用する
4. **反復調整**: 初稿に約2時間、その後細部の調整を繰り返す。花などの複雑なパーツは4回以上描き直すこともある
5. **完成まで**: プロのAA職人は全て手動で5〜6時間かけて1作品を仕上げる

### 部位別テクニック

| 部位 | テクニック |
|------|-----------|
| **目** | 「心」の字は丸みが瞳のカーブにフィットしやすい。「斥」「乍」をまつ毛・瞼の表現に使用 |
| **髪** | 「：」でスクリーントーン的効果。「：：：」と「：．：」で明暗グラデーション。「州」「洲」など画数の多い漢字で濃い部分を表現 |
| **手・指** | 角度によって使える文字が限られるため、元絵のポーズ自体をAA向きに調整することがある |
| **影** | `░` や密度の高い漢字（「鬱」など）を使い分ける |

### AA制作ツール

#### （´д｀）Edit

最も広く使われているAA制作専用エディタ。タブ切り替え式で複数のAAを同時編集可能。

- **画像トレース**: 背景に画像を投影し、上からなぞるようにAAを制作
- **テンプレート**: モナー、ギコ猫、しぃ、忍者、クックルなどの定番キャラをプリセット登録
- **半角/全角スペース可視化**: 不可視文字を色分け表示
- **ブラシツール**: スペースを指定文字で塗るブラシ、直接入力のペンシル、消しゴムの3種
- **矩形選択**: 範囲を矩形で選択し、移動・コピー・反転が可能
- **プレビュー**: 掲示板での実際の表示をシミュレート
- **出力**: BMP/GIF/JPG画像、HTML変換に対応

#### Ascii Art Editor

（´д｀）Edit以前の標準ツール。Windows 98〜XP対応。ピクチャートレーサー、オートトレーサー、リアルタイム2chシミュレータを搭載。

#### その他のツール

| ツール名 | 種別 | 特徴 |
|----------|------|------|
| アスキーアートメーカー | Webツール | ブラウザ上でマウスドラッグによるAA作成 |
| 文字絵エディター | デスクトップ | 画像からAAへの自動変換 |
| AA変換（ツールタロウ） | Webツール | 画像アップロードで即座にAA生成、幅・コントラスト調整可 |
| AAEditor / MoeA | デスクトップ | 画像トレース機能やフォント設定を備えた制作ツール |
| AAMan | デスクトップ | 膨大なAAを管理し、掲示板に素早く貼り付ける管理ツール |

### トレースAA vs オリジナルAA

- **トレースAA**: 既存のイラスト・写真・アニメの画像を元に、AAエディタの画像トレース機能で文字で再現する手法。AA職人いのっくは、フェルメールがカメラ・オブスクラで像をトレースした例を引き、「トレースは絵を描くための一工程・補助道具」と再定義した
- **オリジナルAA**: 元絵を参照せず、AA職人が独自にデザインするAA。モナー、ギコ猫等が代表例
- 現在のAA文化では両者は対立概念ではなく、技法の違いとして共存している

### AAが崩れる問題

AAはMS Pゴシック 16px・行間2pxという特定の表示環境に完全に依存しているため、フォント不一致・OS差異・ブラウザ差異で崩れる。

**対策CSS**:
```css
font-family: 'MS Pゴシック', 'MS PGothic', RobotoJAA, Monapo, 'IPA モナー Pゴシック', '梅Pゴシック', Textar, sans-serif;
```

**代替フォント（非Windows環境向け）**:
- Monapo / Mona（日本語AA専用フォント）
- IPA モナー Pゴシック
- 梅Pゴシック（Ume P Gothic）（Linux向け代替）
- RobotoJAA（クロスプラットフォーム対応）
- Textar（iOS/Android向けWebフォント）

### AA文化の現在

AA職人いのっくは、AA文化の衰退を自然な流れと認識しつつも、希少性による新たな価値を提唱している。名刺やポスターへの活用、現代アートとしての展開を提案し、AAを「現代の枯山水」と表現した。色彩を持たず文字のみで構成されるAAは、白黒写真のように「形に注目がいく」特性を持ち、制約の中での芸術性に価値がある、という見解である。

## やる夫スレ文化

### 概要

やる夫スレは、やる夫を中心にしつつ、2ch AAやアニメ・ゲームキャラのAAを配役して小説や脚本のように物語を進めるスレ文化である。2007年頃から始まり、AA文化の新しい形を確立した。後には「やる夫が出ない作品」も珍しくなく、ジャンルも歴史、学ぶ系、ゲーム再構成、安価、オリジナル長編まで広がった。

### AA長編板との違い

| 項目 | AA長編 | やる夫スレ |
|------|--------|-----------|
| 世界観 | 2ch AAキャラそのものの世界で物語を作る | やる夫を起点に外部作品や歴史人物まで配役 |
| 配役の自由度 | AAキャラに限定 | アニメ・ゲームキャラのAAも「役者」として使用 |
| 全盛期 | 2000年代前半 | 2000年代後半以降 |

### 配役文化（スターシステム）

既存のアニメ・ゲームキャラのAAを「役者」として扱い、本来の設定とは異なる役割を演じさせる独特の形式。

### ジャンル分類

| ジャンル | 特徴 |
|----------|------|
| 学ぶ系（解説系） | やる夫が複雑な事象を対話形式で解説。教育的コンテンツとして爆発的にヒット |
| ゲーム再構成 | DQ、FFなどのRPG原作をやる夫世界で再構成 |
| 歴史系 | 日本史・世界史を題材とした長編 |
| 安価・あんこ系 | 掲示板のレス（安価）やダイスの目（あんこ）で物語展開を決める参加型 |
| オリジナル長編 | 原作なしの独自世界 |

### 代表作品

**学ぶ・勉強系**:
- やる夫が銀行員になるようです（全8話）
- やるやらで学ぶゲームの歴史（全16話+）
- やる夫はアフリカで奇跡を起こすようです
- やる夫で学ぶ「太平記」

**ゲーム再構成**:
- やる夫がドラゴンクエスト5の主人公のようです

**歴史系**:
- やる夫が徳川家康になるようです

**スポーツ系**:
- やる夫はプロ野球選手になるようです（全11話+）

**コメディ・日常系**:
- キッチン【やらない-O】（全18話+）
- サバイバルやる夫（全17話）

**ファンタジー・冒険系**:
- やる夫が空を目指すようです
- やる夫に残された時間は24時間のようです

**参加型**:
- やる夫達は安価で作られた世界で生きているようです

### AA長編板の代表作品

- **世界制服を企むモララー**: AA長編板の老舗リレースレ
- **モララーのビデオ棚**: 長寿シリーズ、派生キャラの出典元
- **しぃのアトリエ ～モナーブルグの錬金術師～**: RPG/ファンタジー系AA長編の代表格

### 商業的影響

- やる夫スレの一部は後にライトノベルとして書籍化
- やる夫スレで培われた物語構成の技術は『ゴブリンスレイヤー』などのライトノベル作家を輩出
- AAキャラクター（モナー・ギコ猫等）の商標登録を企業が試みたことで「AAは誰のものか」という議論が発生。現在では「AAは誰のものでもない、誰でも使って良い」という認識が主流

## Flash動画文化との連携

2000年代前半の「Flash黄金時代」において、AAキャラクターは動画の主役として爆発的な人気を博した。

- **ナイトオブファイヤー (NIGHT OF FIRE)**: AAキャラたちがパラパラを踊るシュールな動画
- **恋のマイアヒ**: 空耳歌詞とAAキャラ「のまネコ」を組み合わせた社会現象的ヒット
- **巫女みこナース**: 電波ソングにカオスなAAアニメーションをつけた動画
- **しいのうた**: しぃのFlash作品。泣ける作品として国内外で絶賛
- **紅白Flash合戦**: 年末に開催された職人たちの祭典

Flash文化はニコニコ動画における「歌ってみた」や「MAD動画」の直接的なルーツとなった。

## 海外への影響

- 英語圏では日本のAAは「Shift_JIS art」として認識され、MS PGothic前提の文字幅で崩れずに見える点が特徴として紹介されている
- 海外・非Windows環境向けにAAを正しく表示するための**Mona Font**が整備された（名前はモナーに由来）
- 英語圏匿名掲示板文化では **2channel → Futaba Channel → 4chan** という系譜が認識されている
- クマーは英語圏で見た目が流用され「Pedobear」として広まった（意味は大きく変質）
