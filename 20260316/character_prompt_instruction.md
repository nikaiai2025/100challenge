## ステップ１：セリフ・語りの抽出

各キャラクターについて、以下の観点で**代表的なセリフや語り口を10〜20文**抽出してください。

- そのキャラクターらしい**口調・言い回し**が出ているもの
- **感情・価値観・世界観**が滲み出ているもの
- 会話AIとして**再現しやすい**もの（独白・問いかけ・断言・皮肉など）
- 地の文（ナレーション的な独白）も含めてよい

**抽出形式：**
```
【キャラ名】
1. 「〜〜〜〜〜」（文脈：〇〇する場面）
2. 「〜〜〜〜〜」（文脈：〇〇する場面）
...
```

## ステップ２：人格プロンプト作成

抽出したセリフをもとに、各キャラクターを**ChatBot（LLM）に演じさせるためのシステムプロンプト**を作成してください。

### プロンプトの必須要素

以下の項目を必ず含めてください：

1. **キャラクター設定**
   - 名前・作品・作者
   - 人物像・背景（2〜3文）

2. **話し方の特徴**
   - 語尾・一人称・よく使う表現
   - 敬語/口語/古語のレベル
   - 特有の言い回しや癖

3. **価値観・世界観**
   - 何を大切にしているか
   - 何を嫌うか・恐れるか
   - 人間・世界への見方

4. **会話上の振る舞い指示**
   - 質問への答え方のスタイル
   - 感情表現の仕方
   - してはいけないこと（キャラを壊す行動）

5. **口調サンプル（Few-shot）**
   - ステップ１で抽出した**全セリフをそのまま全量掲載**する（省略不要）

### プロンプトの出力形式

各キャラクターのプロンプトは**日本語で**、以下の形式で出力してください：

```
=== 【キャラ名】人格プロンプト ===

あなたは〜〜〜〜です。
...

【話し方の特徴】
...

【価値観・世界観】
...

【会話の振る舞い】
...

【口調サンプル（ステップ１の全抽出セリフ）】
- 「〜〜〜」（文脈：〇〇）
- 「〜〜〜」（文脈：〇〇）
※ ステップ１で抽出した10〜20文を全て列挙すること
```

## 注意事項

- 各キャラクターの**原文の口調を最大限尊重**してください
- プロンプトはLLMのシステムプロンプトとして**そのままコピーして使える**品質にしてください
- キャラクターを現代人として動作させる場合も、**話し方の雰囲気は原文に忠実**に保ってください
- **ネタバレを含む内容**（結末・死など）は人格プロンプトには含めず、あくまでキャラの生きている状態での人格を設定してください
